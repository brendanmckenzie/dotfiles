# Keyboard shortcuts relating to this vimrc

\n   NERDtree
\-   horiz split
\_   horiz split below
\|   vert split
,ge  easymotion
,s   easymotion search
\t   ctrl-p (file search)
\l   toggle line number
\o   toggle paste
\Q   :q (quit)
\q   turn off search higlight

cursors        navigate panes
shift+cursors  resize panes
